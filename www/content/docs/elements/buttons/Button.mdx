Primary buttons should be used for the primary desired user action. Do not use 2
primary buttons in the same screen. If there are multiple parimary options, use
secondary-gray.

<Playground hideToggle={true}>
  <Button>Purchase</Button>
</Playground>

Our secondary-outline is our default secondary button style. Avoid using this in
the presence of inputs, as to not confuse itâ€™s use.

<Playground hideToggle={true}>
  <Box>
    <Button width="33%">Purchase</Button>
    <Spacer mb={1} />
    <Button variant="secondaryGray" width="33%">
      Make offer
    </Button>
  </Box>
</Playground>

In the presence of inputs, use secondary-gray for the secondary button style.

      <BorderBox mb={1}>Input text</BorderBox>
      <BorderBox mb={1}>Input text</BorderBox>
      <Button variation="secondary">Add another</Button>
    </Flex>
  </BorderBox>
</Playground>
