## Usage guidelines

Inputs should be used in instances where we require users to enter information that requires explanation or context. These usages tend to be more complex and therefore require structured inputs with a variety of contextual info. For light-weight applications we recommend using the  [quick-input](https://palette.artsy.net/elements/inputs/quickinput/) instead.   

## Input only

<Playground>
  <BorderBox>
    <Input placeholder="Your offer" />
  </BorderBox>

<Playground>

## Input + Title + Description

<Playground>
