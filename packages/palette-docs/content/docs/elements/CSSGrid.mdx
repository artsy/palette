---
name: CSSGrid
---

The CSSGrid component is a layout container for content that exposes props
related to CSS grid.

Palette also offers a [Grid element](/elements/layout/grid/), which can be used
for layouts based on a Bootstrap-style twelve column grid.

### A Simple CSSGrid

<Playground>
  <CSSGrid gridTemplateColumns="repeat(2, 1fr)" gridGap={2} my={2}>
    <Box>This grid displays...</Box>
    <Box>...two columns...</Box>
    <Box>...per row...</Box>
    <Box>...with a gap of 20px.</Box>
  </CSSGrid>
</Playground>

### A Responsive CSSGrid

<Playground>
  <CSSGrid
    gridTemplateColumns={["repeat(2, 1fr)", "repeat(4, 1fr)"]}
    gridGap={[2, 4]}
    my={2}
  >
    <Box>This grid displays...</Box>
    <Box>...two columns at xs...</Box>
    <Box>...but 4 columns at sm or larger.</Box>
    <Box>The gap is also different.</Box>
  </CSSGrid>
</Playground>
