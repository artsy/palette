name: HTML
source: elements/HTML
---

`HTML` renders the given HTML string and wraps it in a `<Text>` component. All
supported text variants can be passed in.

<Playground>
  <HTML
		variant='sm'
    size={4}
    html={"<h1>H1 Headline</h1><p>Paragraph tag</p><p>Paragraph tag</p>"}
  />
</Playground>
