name: Clickable
source: elements/Clickable
---

Clickable is a utility component useful for wrapping things like `div`s without
having to deal with the requirements to make the `div` accessible. It is the
web-equivalent of `TouchableWithoutFeedback`. It supports the same system props
as `<Box />`

<Playground>
  <Clickable bg="black5" p={2} onClick={() => alert("Clicked")}>
    <Text variant="lg-display">Click Me</Text>
  </Clickable>
</Playground>
