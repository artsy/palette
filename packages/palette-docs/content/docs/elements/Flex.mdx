### Basics

Flex is simply a `Box` component with `display="flex"` set by default.

<Playground>
  <Flex flexDirection="column" bg="black5" p={2}>
    <Text variant="sm-display">
      A Flex component is a layout container for content that exposes all props
      related to flexbox.
    </Text>
  </Flex>
</Playground>

### Flex Item Order

Flex items are displayed in the same order as they appear in the source document
by default. We can use the `order` property to change the ordering.

<Playground>
  <Flex bg="black5" p={2}>
    <Flex order="2" color="red100">
      <Text variant="sm">Item 2</Text>
    </Flex>
    <Flex order="1" color="blue100">
      <Text variant="sm">Item 1</Text>
    </Flex>
  </Flex>
</Playground>
