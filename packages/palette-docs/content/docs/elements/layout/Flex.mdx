## Flex Item Order

Flex items are displayed in the same order as they appear in the source document
by default. We can use the `order` property to change the ordering.

<Playground>
  <Flex>
    <Flex order="2">Item 2</Flex>
    <Flex order="1">Item 1</Flex>
  </Flex>
</Playground>
