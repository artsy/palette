import _ from "lodash"

The Grid element is a layout container for rendering a Bootstrap-style twelve
column grid.

See [styled-bootstrap-grid](https://github.com/dragma/styled-bootstrap-grid) for
full documentation.

Palette also offers a [CSSGrid element](/elements/layout/cssgrid/), which can be
used for layouts based on the `display:grid` CSS feature.

### Examples

<Grid>
  <Row>
    {_.times(12, index => {
      return (
        <Col sm={1} key={index}>
          <BorderBox>{index + 1}</BorderBox>
        </Col>
      )
    })}
  </Row>
</Grid>
