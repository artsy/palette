Use a bar chart to visualize a distribution of 1 dynamic y-axis variable over 1
consistent x-axis variable.

The extent of interaction on this component is a tool-tip that reveals on hover,
showing data for that bar. This graph is intended to show relativity to itself,
not relativity to other graphs.
