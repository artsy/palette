### Simple

<Playground>
  <EntityHeader name="Francesca DiMattio" />
</Playground>

### With image

<Playground>
  <EntityHeader
    imageUrl="https://picsum.photos/110/110/?random"
    name="Francesca DiMattio"
  />
</Playground>

### Fallback with initials

When no `imageURL` is provided it will defer to the `initials` prop:

<Playground>
  <EntityHeader
    initials="FD"
    name="Francesca DiMattio"
    meta="American, b. 1979"
    href="http://www.artsy.net/artist/francesca-dimattio"
  />
</Playground>

### Smaller version

When `smallVariant` prop is provided:

<Playground>
  <EntityHeader
    smallVariant
    initials="FD"
    name="Francesca DiMattio"
    imageUrl="https://picsum.photos/110/110/?random"
    href="http://www.artsy.net/artist/francesca-dimattio"
  />
</Playground>
