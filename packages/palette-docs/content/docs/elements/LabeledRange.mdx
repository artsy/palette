---
name: LabeledRange
---

All `LabeledRange` props are passed to the underlying [Slider](#/docs-slider)
component.

### Default

<Playground>
	<Box p={2}>
		<LabeledRange
			min={1}
			max={100}
			step={1}
			defaultValue={[1, 100]}
			disabled={false}
			label="Height"
		/>
	</Box>
</Playground>

### With unit defined

<Playground>
	<Box p={2}>
		<LabeledRange
			min={1}
			max={100}
			step={1}
			defaultValue={[1, 100]}
			disabled={false}
			label="Height"
		/>
	</Box>
</Playground>

### With formatter function defined

<Playground>
	<Box p={2}>
		<LabeledRange
			min={1}
			max={100}
			step={1}
			defaultValue={[1, 100]}
			disabled={false}
			label="Price"
			formatter={(min, max, maxIndicator) => {
				return "$" + min + " - $" + max + maxIndicator
			}}
		/>
	</Box>
</Playground>
