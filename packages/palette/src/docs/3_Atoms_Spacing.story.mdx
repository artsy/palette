# Spacing

The Artsy spacing system employs a base-10px shorthand for working with margins
and padding, meaning `1 = 10px`, `2 = 20px`, `.5 = 5px`, and so on.

We utilize [styled-system's `space` API](https://styled-system.com/api/#space) throughout all of Palette.
