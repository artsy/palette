# Grid

Using a responsive 12-column grid for all surfaces. Looking to the breakpoints, we build a fluid system between (Desktop) 1920px — 769px and (Tablet / Mobile) 768px — 375px. The 12 columns are always consistent as they are a construct of the code, though for Tablet / Mobile screens, we keep content to 6-column containers as shown below.
