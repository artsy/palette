# Spacing

The Artsy spacing system employs a base-10px shorthand for working with margins
and padding, meaning `1 = 10px`, `2 = 20px`, `.5 = 5px`, and so on.

We utilize [styled-system's `space` API](https://styled-system.com/api/#space) throughout all of Palette.

#### Example

<Canvas isExpanded>
	<BorderBox p={2}><Text>This container has 20px padding around it</Text></BorderBox>
	<BorderBox p={4}><Text>This container has 20px padding around it</Text></BorderBox>
</Canvas>

#### All available spacing values

<Canvas>
	<Flex mt={4}>
		<Join separator={<Spacer mx={2} />}>
			{Object.keys(THEME.space).sort((a, b) => parseFloat(a) - parseFloat(b)).map((key) => {
				const px = parseFloat(key) * 10
				return (
					<Flex key={key} flexDirection='row'>
						<Text variant="lg" pr={.5}>{key}</Text>
						<Text variant="xs">{px}px</Text>
						<Box width={px} height={1} bg="black60" />
					</Flex>
				)
			})}
		</Join>
	</Flex>
</Canvas>
